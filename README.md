# TaskListApp

Projeto criado com foco em estudo e prática de desenvolvimento backend.  
O objetivo principal é consolidar conceitos como estruturação de API, organização em camadas, CRUD e persistência de dados.

## Propósito do projeto

TaskListApp implementa um sistema básico de gerenciamento de tarefas, permitindo criar usuários e associar tarefas a eles.  
O projeto serve como material de estudo e fixação de conteúdo, não sendo voltado para produção.

Principais pontos estudados:

- Estruturação de API REST
- Padrão em camadas
- CRUD com entidades relacionadas
- Boas práticas iniciais de desenvolvimento backend

## Tecnologias utilizadas

- .NET / C#
- Entity Framework Core
- Banco de dados relacional
- API REST
